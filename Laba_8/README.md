# Лабораторная работа №8 — Beans, JSP (Вариант 46)

Web-приложение из трёх JSP-страниц с Bean-компонентом для обмена данными между страницами. Вычисляет функцию из лабораторной работы №1 — разделение последовательности чисел на нечётные/чётные позиции.

## Вариант 46

Число варианта: **50** → двоичное: `110010`

Особенность: П.5-П.8 по 1 биту, П.9 — два последних бита (значения 0-3).

| Пункт | Значение | Что означает |
|-------|----------|--------------|
| П.5 | **1** | Программный код функции — **в классе Bean-компоненте** |
| П.6 | **1** | Заголовки страниц — **«Раз», «Два», «Три»** |
| П.7 | **0** | Стартовая страница — **текст задания + ФИО + ссылка** для перехода |
| П.8 | **0** | Финишная страница — **строки одна за другой** (таблица без видимых границ) |
| П.9 | **2** (binary `10`) | При повторном переходе на Главную — **изменение счётчика, размещённого на Главной странице** |

## Задача (из лабы №1)

Разделение последовательности целых чисел на две подпоследовательности:
- **Нечётные позиции** (1, 3, 5, ...) — первый, третий, пятый элементы
- **Чётные позиции** (2, 4, 6, ...) — второй, четвёртый, шестой элементы

### Bean: Calculator.java

Класс `jspappl.Calculator` реализует `Serializable`. Хранит данные между страницами через `scope="session"`.

| Свойство | Тип | Описание |
|----------|-----|----------|
| `fio` | `String` | ФИО студента (задаётся на jsp_1.jsp) |
| `input` | `String` | Введённые числа (передаются с jsp_2 в jsp_3) |
| `oddPositions` | `List<String>` | Числа на нечётных позициях (1, 3, 5, ...) |
| `evenPositions` | `List<String>` | Числа на чётных позициях (2, 4, 6, ...) |
| `bad` | `List<String>` | Невалидные токены |

При вызове `setInput()` автоматически срабатывает `compute()` — реализация П.5=1 (логика в Bean).

### Жизненный цикл сессии

```
1. Пользователь открывает /lab8/jsp_1.jsp
   → Tomcat создаёт сессию
   → <jsp:useBean> создаёт новый Calculator в сессии
   → <jsp:setProperty fio="..."> устанавливает ФИО

2. Пользователь кликает «Перейти на Главную»
   → /lab8/jsp_2.jsp
   → <jsp:useBean> находит существующий Calculator в сессии (не создаёт новый!)
   → ++counter (статический счётчик в сервлете JSP)
   → Форма с полем ввода

3. Пользователь вводит "10,20,30" и отправляет
   → POST /lab8/jsp_3.jsp?nums=10,20,30
   → <jsp:useBean> — тот же Calculator
   → <jsp:setProperty input param="nums"> вызывает setInput("10,20,30")
   → setInput() автоматически вызывает compute()
   → Bean заполняет oddPositions=[10,30], evenPositions=[20]
   → Страница выводит результаты в таблице без границ

4. Пользователь кликает «Возврат» → jsp_2.jsp
   → Счётчик увеличивается (теперь = 2)
   → Поле ввода предзаполнено предыдущим input
```

## Сборка и развёртывание

Предполагается что Tomcat 9 уже установлен и настроен на порт 9090 (см. Laba_7).

### Компиляция Bean

```bash
cd Laba_8/src/main/java
javac -encoding UTF-8 -cp ../../../lib/javax.servlet-api-4.0.1.jar jspappl/Calculator.java
```

### Сборка WAR

```bash
cd Laba_8
rm -rf build
mkdir -p build/WEB-INF/classes/jspappl
cp src/main/java/jspappl/Calculator.class build/WEB-INF/classes/jspappl/
cp src/main/webapp/*.jsp build/
cd build && jar cvf ../lab8.war . && cd ..
```

### Деплой

```bash
cp lab8.war /opt/homebrew/opt/tomcat@9/libexec/webapps/
```

Tomcat автоматически распаковывает WAR. Если Tomcat не запущен:

```bash
/opt/homebrew/opt/tomcat@9/bin/catalina run
```

## Использование

Открыть в браузере:

```
http://localhost:9090/lab8/jsp_1.jsp
```

1. На стартовой странице **«Раз»** — текст задания, ФИО, ссылка
2. Кликнуть «Перейти на Главную страницу»
3. На странице **«Два»** — счётчик (растёт при каждом возврате), форма ввода чисел
4. Ввести числа, например `10, 20, 30, 40, 50`, нажать «Вычислить»
5. На финишной странице **«Три»** — результаты разделения на позиции
6. «Возврат» — обратно на «Два», счётчик увеличился

## Соответствие ТЗ (Вариант 46)

| Пункт ТЗ | Как выполнено |
|----------|---------------|
| 2. Три JSP-страницы + Bean | jsp_1.jsp, jsp_2.jsp, jsp_3.jsp + Calculator.java |
| 3. Вычисление функции из лабы №1 на «Главной странице» | Форма на jsp_2.jsp, обработка в Calculator при переходе на jsp_3 |
| 4. Параметры через текстовое поле, результат на «Финишной» | `<input name="nums">` → `setProperty input` → результаты на jsp_3 |
| 5. Код функции **в Bean** (П.5=1) | `Calculator.compute()` |
| 6. Заголовки **«Раз», «Два», «Три»** (П.6=1) | `<title>Раз</title>`, `<h1>Раз</h1>` и т.д. |
| 7. Стартовая: задание + ФИО + **ссылка** (П.7=0) | `<a href="jsp_2.jsp">Перейти на Главную</a>` |
| 8. Финишная: строки одна за другой (П.8=0) | `<table>` без `border`, один столбец |
| 9. Счётчик **на Главной странице** (П.9=2) | `<%! int counter; %>` + `<% counter++; %>` в jsp_2.jsp |
| 10. Инструменты автоматизации | Bean-компонент с геттерами/сеттерами (можно создать через «Инкапсулировать поля» в IDE) |
