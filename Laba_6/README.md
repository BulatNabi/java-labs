# Лабораторная работа №6 — Потоки. Графика (Вариант 66)

Приложение на Java Swing с использованием потоков (Thread). Объекты (текстовые надписи) перемещаются в окне, отскакивая от стен.

### Классы

| Класс | Описание |
|-------|----------|
| `Main` | Точка входа. Создаёт `DemoWindow` и `ControlWindow` в EDT (SwingUtilities) |
| `MovingObject` | Реализует `Runnable`. Каждый объект — отдельный поток. Хранит координаты (x, y), вектор скорости (vx, vy), цвет, текст, номер. Отскакивает от границ окна. При изменении размера окна — корректирует позицию |
| `DemoWindow` | `JFrame` с `JPanel`. **Resizable = true** (п.10=2). Хранит список объектов, предоставляет методы `addObject()`, `findByNumber()`, `getFieldWidth/Height()` |
| `ControlWindow` | `JFrame` с элементами управления. Два раздела: запуск нового ФиО и изменение существующего. Закрытие любого окна завершает приложение (`EXIT_ON_CLOSE`) |

### Потоки

Каждый `MovingObject` работает в собственном daemon-потоке. Цикл:
1. Получить текущие размеры `DemoPanel`
2. Рассчитать смещение с учётом `dt` (время между кадрами)
3. Проверить столкновения со стенами — отразить вектор скорости
4. Запросить перерисовку (`repaint()`)
5. `Thread.sleep(5)` — ~200 FPS

## Запуск

### Компиляция и запуск
```bash
cd Laba_6/src/main/java
javac -encoding UTF-8 org/example/*.java
java org.example.Main
```

### Использование

1. В **Управляющем окне** введите текст ФиО (например, "Привет"), номер, выберите цвет из списка, укажите скорость в пикс/с
2. Нажмите **Пуск** — объект появится в левом верхнем углу ДО и полетит в случайном направлении
3. Максимум 5 объектов (задано в коде)
4. Для изменения существующего объекта: введите его номер, выберите новый цвет и скорость, нажмите **Применить**
5. Попробуйте изменить размер ДО — объекты адаптируются к новым границам

## Соответствие ТЗ (Вариант 66)

| Пункт ТЗ | Как выполнено |
|-----------|---------------|
| 1. Число ФиО задано в коде УО | `MAX_OBJECTS = 5` в `ControlWindow` |
| 2. ФиО — объекты (надписи) | `MovingObject` рисует текст через `g.drawString()` |
| 3. Цвет — выпад. список (5+) | `JComboBox` с 7 цветами: Красный, Синий, Зелёный, Жёлтый, Чёрный, Оранжевый, Розовый |
| 4. Выбор запускаемого ФиО из текст. поля | `JTextField textField` — ввод текста надписи |
| 5. Нач. скорость из текст. поля | `JTextField speedField` — ввод скорости в пикс/с |
| 6. Выбор запущенного ФиО из текст. поля | `JTextField selectNumberField` — ввод номера |
| 7. Номер вручную из УО | `JTextField numberField` — ручной ввод, проверка уникальности |
| 8. Смена номера — нет | Поле «Новый номер» отсутствует в УО |
| 9. Скорость из выпад. списка (5 скоростей) | `JComboBox adjustSpeedCombo`: Очень медленно / Медленно / Средне / Быстро / Очень быстро |
| 10. Изменение размера окна — да | `setResizable(true)`, объекты clamp'ятся в новые границы |
| Запуск из верхнего левого угла | `x = 5, y = 25` в конструкторе `MovingObject` |
| Случайное направление | Угол генерируется `Random` в диапазоне (0, π/2) |
| Отражение от стен | «Угол отражения = угол падения»: инвертируется компонент vx или vy |
| Номер рядом с ФиО | Отображается как `(N) текст` |
| Уникальность номеров | Проверка `demo.numberExists()` перед добавлением |
| Закрытие любого окна | Оба окна: `setDefaultCloseOperation(EXIT_ON_CLOSE)` |
